# Analysis of Prompts Project

A research project focused on analyzing and improving prompt engineering through meta-cognitive frameworks and structured thinking approaches.

## Project Structure

```
.
├── src/                    # Source code
│   ├── mojo/              # Mojo source files
│   └── python/            # Python integration files
├── config/                # Configuration files
│   ├── cursorrules/       # Cursor rule configurations
│   ├── docker-compose.yml # Docker compose configuration
│   ├── Dockerfile        # Docker configuration
│   ├── pixi.toml         # Pixi configuration
│   └── requirements.txt   # Python dependencies
├── data/                  # Data files
│   ├── raw/              # Raw data
│   └── processed/        # Processed data
├── docs/                  # Documentation
│   ├── research/         # Research documents
│   ├── analysis/         # Analysis documents
│   ├── guides/           # How-to guides
│   └── reports/          # Project reports
├── utils/                # Utility scripts
├── examples/             # Example code
└── tests/                # Test suites

## File Naming Conventions

### Prefixes
- `meta-`: Meta-analysis related files
- `cognitive-`: Cognitive models and processes
- `framework-`: Conceptual frameworks
- `prompt-`: Prompt engineering
- `analysis-`: In-depth analysis
- `model-`: Models and algorithms
- `tool-`: Tools and utilities
- `config-`: Configuration files
- `data-`: Data files
- `docs-`: Documentation

### Suffixes
- `-v[0-9]+`: Version number
- `-analysis`: Analysis documents
- `-model`: Model descriptions
- `-framework`: Framework documents
- `-guide`: Instructional guides
- `-report`: Reports
- `-notes`: Notes and brainstorming

## Getting Started

1. Clone the repository
2. Install dependencies from `config/requirements.txt`
3. Follow setup instructions in `docs/guides/`
